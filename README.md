# 💎 Glamm Fashion

Glamm Fashion is a modern and responsive jewellery e-commerce website built with **HTML, CSS, and JavaScript**, powered by **Firebase** for authentication, Firestore database, and storage.

## ✨ Features

* Responsive design
* Dynamic product listing
* Category management
* Hero banner management
* Firebase Authentication
* Cloud Firestore integration
* Firebase Storage for images
* Admin dashboard
* Search and filtering
* Wishlist and cart functionality
* Mobile-friendly UI

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Firebase Authentication
* Cloud Firestore
* Firebase Storage
* Firebase Cloud Functions

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/glamm-fashion.git
```

2. Open the project folder

```bash
cd glamm-fashion
```

3. Install dependencies (if applicable)

```bash
npm install
```

4. Run the project

```bash
npm run dev
```

or open `index.html` using a local server.

## 🔥 Firebase Configuration

Configure your Firebase project and add your credentials in:

```
assets/js/config/firebase.js
```

Enable:

* Authentication
* Cloud Firestore
* Firebase Storage
* Cloud Functions
